Vishnu Hackathon

Mr.Abinav Dayal

Full stack development with Django

We would be discussing following topics in order:

  • Basics of Internet,Web App Stack
  • CSS with Bootstrap, JavaScript with jQuery
  • AJAX, Django and MySQL

We would like the students to have following traits:

  1. Decent knowledge of python and basic coding skills
  2. Be enthusiastic.
  3. Be willing to innovate.
  4. Be eager to learn by exploring and asking good questions.
  5. Be willing to work hard. And finally not complain about anything.

Following is the detailed program:

Day1 Basics of Internet:

  1. We shall discuss how the web works and how web applications operate.

    2. This will give a good context to entire course.

  2. Structure of web page.

    3. This will discuss HTML in detail.

  3. Web page in STYLE.

    4. This will discuss CSS in detail along with BootStrap the popular CSS Framework.

Mini hackathon to build a page and style it.

Day2 JavaScript:

  1. Adding behavior to a page, DOM, manipulating page using code.
  2. jQuery: making your task of DOM manipulation a piece of cake.
  3. AJAX using jQuery and JSON and consuming a web service.
  4. Creating a small game using HTML, CSS and JavaScript.

Day ends with another mini hackathon to have people innovate in whatever they want to make.

Day3 Django:

  1. Introducing web server and starting a Django app.
  2. Databases MySQL working with database in Django.
  3. Make Instagram clone in Django using whatever we learnt till now.

Day ends with a full night hackathon asking people in teams to build an end to end app of there choice.
